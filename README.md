# Full-Stack Repo 🚀

This repository is a simple workspace for my full-stack learning journey.
It currently focuses on building a basic understanding of development tools and concepts, starting with Git.

Even with basic commands, it allows creating branches, experimenting safely, and reverting to earlier versions when needed. I’m also looking forward to learning how it is used in real projects for teamwork, code reviews, and handling larger codebases.
